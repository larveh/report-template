# Report template

This is a template for making reports in Pandoc. Write your report in Markdown and export it to PDF. 
For convenience, docker is used to run Pandoc locally.

Supports:

* embedding images with figure numbering
* References from .bib
* code blocks

Dependencies:

* docker
* pandoc/extra
